Política de Privacidade

Última atualização: Setembro/2025

O aplicativo Motivário tem como objetivo oferecer frases motivacionais personalizadas, exibidas no feed, notificações push e widgets (caso o usuário permita). Sua privacidade é muito importante para nós.

1. Dados que coletamos

Coletamos apenas as seguintes informações fornecidas pelo usuário:

  • Nome (opcional)
  • Idade
  • Gênero

Esses dados servem exclusivamente para personalizar as frases exibidas no aplicativo. Caso o usuário crie uma conta, também poderemos armazenar as frases marcadas como favoritas, utilizando o serviço Supabase.

2. Como usamos seus dados

Usamos os dados coletados para:

  • Personalizar o conteúdo exibido.
  • Manter sua conta e salvar favoritos (se você optar por criar uma conta).
  • Enviar notificações, caso você permita.

Não vendemos, compartilhamos ou usamos seus dados para publicidade.

3. Armazenamento e segurança

Os dados são armazenados de forma segura em nossos sistemas. Adotamos medidas técnicas adequadas para proteger suas informações contra acesso não autorizado.

4. Pagamentos

As assinaturas Premium são processadas diretamente pelas lojas Google Play Store e Apple App Store. Não temos acesso e não armazenamos dados de pagamento.

5. Direitos do usuário

De acordo com a LGPD, você pode:

  • Acessar os dados coletados.
  • Corrigir ou excluir seus dados.
  • Cancelar sua conta a qualquer momento.

Para exercer seus direitos, entre em contato: contato@motivario.app

6. Alterações

Podemos atualizar esta política periodicamente. Caso ocorram mudanças relevantes, avisaremos no aplicativo.

7. Contato

Dúvidas? Entre em contato: contato@motivario.app
